RustForge

Automated Painting Guide

Follow these simple steps to start painting your images onto Rust signs instantly.

Step-by-Step Instructions

STEP 1
Load Your Image

Use the main RustForge application window to load your desired image file by clicking the "Load Image..." button.

STEP 2
Capture Canvas Region

In-game, use the Canvas hotkey (F9 default) to draw a box around the entire in-game canvas where the painting will occur.

Press Enter to save the selection.
STEP 3
Capture Palette Region

Use the Palette hotkey (F10 default) to draw a box only around the in-game color palette. This samples the available colors.

NOTE: Only capture the color palette area!
STEP 4
Configure Settings

Open the settings window to select your preferred painting method (e.g., VoidWeave), apply filters, and adjust the paint speed for optimal results.

STEP 5
Start Painting

Once your image is processed and both the Canvas and Palette regions are correctly defined, press the Start hotkey to begin the automated painting process.

F11 default
STEP 6
Pause and Stop

Use the assigned hotkeys to pause or stop the painting process at any time to regain control of your mouse.

F11 to Pause/Resume, Escape to Stop

Default Hotkeys Reference

F9: Canvas Overlay Toggle

Toggles the canvas selection overlay to define the painting area.

F10: Palette Overlay Toggle

Toggles the palette selection overlay to define the color palette area.

F11: Start / Pause

Starts the painting process or pauses a running session.

Escape: Stop

Immediately stops the painting process.

F5: Resume On Click

Click a point on the canvas with your image loaded to resume the paint session from that row.

Hotkeys are fully configurable in the application's settings window.
